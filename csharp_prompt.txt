## Project Overview
Create a C# project that includes a simple **Calculator** class with basic arithmetic operations and **unit tests** using xUnit.

## Requirements
- Implement a **Calculator** class with methods for:
  - Addition (`Add`)
  - Subtraction (`Subtract`)
  - Multiplication (`Multiply`)
  - Division (`Divide` - ensure it handles division by zero correctly)
- Write **unit tests** using `xUnit` to validate all Calculator methods.
- Ensure proper **exception handling** in the division method.
- Follow **clean coding principles**, use **camelCase** for variables, and **PascalCase** for methods.

## Test Cases (Using xUnit)
- Verify that `Add(2, 3)` returns `5`.
- Verify that `Subtract(7, 2)` returns `5`.
- Verify that `Multiply(4, 5)` returns `20`.
- Verify that `Divide(10, 2)` returns `5`.
- Ensure that `Divide(10, 0)` throws a `DivideByZeroException`.

## Additional Guidelines
- Use **dependency injection** if applicable.
- Follow **SOLID principles** where relevant.
- Generate documentation comments (`/// <summary>...</summary>`) for all public methods.
- Ensure **high test coverage** with edge cases.
